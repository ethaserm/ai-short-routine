How it all works

read this first — simple explanation of the whole process

🎯
What you're making
the big picture
You're making a 50–60 second YouTube Short entirely with free AI tools. No camera, no filming, no editing skills needed. Just prompts and a bit of time.
🖼️
Step 1 — The Images
what the viewer actually sees
You start by describing what you want to see in each scene. Nano Banana 2 (on Runway) then generates a still image for each scene.

For a 60 second video you need about 8–10 images — one per scene. You use a prompt in the Script & Prompts tab to generate all the image descriptions automatically.
🎬
Step 2 — The Video Clips
making the images move
Still images don't move — so you upload each one to a video AI like Luma, Kling, or Pika and it animates it into a 5–6 second moving clip. You rotate between sites so you don't run out of free credits.

You use a prompt to tell the AI how to animate each image (e.g. slow zoom, camera pan, wind blowing).
🎙️
Step 3 — The Voice (voiceover)
the words spoken over the video
The script is only the words that get spoken — like a narrator talking over the clips. You generate it with Claude or Gemini. It has 8–10 sentences, each about 5–6 seconds long, adding up to ~60 seconds total.

You paste the script into ElevenLabs which turns it into a real-sounding AI voice. That becomes your audio file.
🎵
Step 4 — Putting it all together
the final edit in capcut
Open CapCut (free). Import all your video clips and your voiceover audio. Line the clips up so they cover the audio. Add auto-captions, background music, and colour grade. Export at 1080×1920 (vertical, for Shorts). Done.
💡
The simple order to follow
do these tabs in this order
1. Channel Prompts tab → find your niche and plan your channel
2. Script & Prompts tab → generate your script, image descriptions, and animation prompts
3. Routine tab → follow step by step to actually make the video
4. Quick Links (bottom of Routine tab) → shortcuts to every website you need

Script & Prompts

use these every time you make a new video · do them in order

Do these 3 steps in order for every video:
1. Generate image descriptions (what to draw) → 2. Generate animation prompts (how to make images move) → 3. Generate your voiceover script (the words spoken)
setup
Type your niche once
This fills into all 3 prompts below automatically so you don't have to type it every time.
① Images — generate these first in Nano Banana 2
step 1 Image Prompt Generator
Paste your video topic below the prompt. It gives you 10 image descriptions — paste each one into Runway → Nano Banana 2 to generate your images. Download them all. ⚠️ Always set aspect ratio to 9:16 (vertical) for Shorts!
You are a visual director for AI-generated YouTube Shorts. I will give you a video topic. Generate 10 detailed image generation prompts for scenes that would appear in a short video about this topic. Rules: - One image prompt per scene - Each prompt should describe a cinematic, photorealistic scene - Include: subject, setting, lighting, mood, camera angle - Style: cinematic, 8K, dramatic lighting, ultra detailed, vertical 9:16 aspect ratio for YouTube Shorts - Do not include any text or words in the image - Number each prompt 1 to 10 The niche is: [insert niche] The video topic is: [describe your video topic here e.g. "the rise and fall of ancient Rome"]
② Videos — animate each image into a moving clip
step 2 Video Animation Prompt Generator
Paste your image descriptions below the prompt. It gives you one animation instruction per image — paste these into Luma, Kling, Pika, or Vidu when you upload each image to animate it.
You are a video director for AI-generated short films. I will give you a list of image descriptions. For each one, write a short animation prompt that tells an AI video tool how to bring that image to life. Rules: - One animation prompt per image - Keep each prompt under 20 words - Use cinematic camera movements: slow zoom in, slow pan left, camera drift forward, subtle parallax, gentle push in - Add atmosphere: dust particles, light rays, wind in hair, flickering flame, rippling water - Match the mood of the image - Number each prompt to match the image number The niche is: [insert niche] Here are the image descriptions: [paste your image descriptions here]
③ Voice — generate the script for your voiceover
step 3 Script Generator
Generates voiceover scripts matched to your video scenes. First generate your image descriptions (Step 1), then paste them at the bottom of this prompt so the script matches what's on screen. Paste into Claude or Gemini, then into Speechma for the voiceover.
number of scripts: 10
You are an expert YouTube Shorts scriptwriter. Generate 10 complete voiceover scripts for a YouTube Shorts video in the niche: [insert niche]. Each script must: - Start with a powerful 1 sentence hook that grabs attention in the first 3 seconds - Be 8–10 sentences long (50–60 seconds spoken at natural pace) - Flow naturally as a voiceover — no bullet points, just spoken sentences - Build curiosity or tell a story throughout - End with this exact closing line: "Follow for more [niche topic] they don't want you to know about." - Be written for a general audience, no jargon IMPORTANT: I will paste my scene descriptions below. Write the script so each sentence matches what is happening in each scene in order — the voiceover must be relevant to the visuals. Format each script as: SCRIPT [number]: [title] [full script text] Only output the scripts — no extra commentary. Here are my scene descriptions — write the script to match these in order: [paste your scene descriptions from Gemini here]
🏆 ranking videos — countdown format
ranking Ranking Video Script Generator
Generates a countdown ranking script — each position is one image/clip. Numbers count down on screen while the voiceover reveals each entry. Perfect for dark history, psychology, space, ancient civilisations.
number of scripts: 5
You are an expert YouTube Shorts scriptwriter specialising in ranking videos. Generate 5 complete ranking countdown scripts for a YouTube Shorts channel in the niche: [insert niche]. Each script must follow this exact format: - Hook (1 sentence): A shocking statement that sets up the ranking e.g. "These are the 5 most brutal punishments in history — ranked." - Position 5 (2 sentences): Introduce and describe entry #5 - Position 4 (2 sentences): Introduce and describe entry #4, slightly more shocking - Position 3 (2 sentences): Introduce and describe entry #3, building tension - Position 2 (2 sentences): Introduce and describe entry #2, very shocking - Position 1 (2 sentences): The most shocking reveal — "And the number one most..." - Closing line (1 sentence): A cliffhanger or call to action e.g. "Which one shocked you most? Comment below." Rules: - Each position must be MORE shocking or dramatic than the one before it - Keep each position description short and punchy — 2 sentences max - Write naturally as a voiceover — no bullet points - General audience, no jargon - Total script should be 50–60 seconds when spoken Format each script as: RANKING SCRIPT [number]: [title] [full script] Only output the scripts — no extra commentary.
